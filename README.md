# Unification Simulation

Distance-based railway network simulation for analyzing the integration of the Gyeongui Line.

## Overview

Unification Simulation is a transportation network analysis project that evaluates the impact of integrating the Gyeongui Line railway system.

The project uses railway station and distance data to simulate transportation routes and compare network connectivity before and after railway integration. Through distance-based calculations, the system analyzes transportation accessibility, route efficiency, and infrastructure utilization.

## Objectives

* Simulate railway network integration scenarios
* Analyze station connectivity and transportation routes
* Calculate travel distances across the railway network
* Compare transportation efficiency before and after integration
* Support infrastructure and transportation planning
